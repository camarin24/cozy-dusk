# Cozy Dusk

A simple dark theme for [VSCode](https://code.visualstudio.com/). 

This theme is based on the [Mayukai Theme](https://marketplace.visualstudio.com/items?itemName=GulajavaMinistudio.mayukaithemevsc) by [@GulajavaMinistudio](https://github.com/GulajavaMinistudio/Mayukai-Theme).

Many changes have been made based on nothing special, just personal taste.
